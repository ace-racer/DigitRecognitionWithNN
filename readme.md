##How to run the client application in your machine?

1. Navigate to GUI folder
2. Install the dependencies
    1. Tensorflow
	2. TKinter
	3. Scipy
	4. Matplotlib
3. Unzip the t10k-images-idx3-ubyte.gz file in the MNIST-data folder once the files are downloaded
